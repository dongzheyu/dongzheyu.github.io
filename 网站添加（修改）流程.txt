网站添加（修改）流程：
1.写代码，vue格式，放在/src/views/文件夹下，HTML语法，可参照其他文件
2.添加路由，在src/router/index.ts文件，和对应的展示文件下的路由信息，根据内容添加路由
3.npm调试，运行开发测试命令:npm run dev
4.测试成功后，构建:npm run build
5.添加到开源仓库:git add .
                           git commit -m '提交信息'
                           git push
