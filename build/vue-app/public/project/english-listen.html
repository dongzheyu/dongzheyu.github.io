English Listen - 专业英语听写训练工具 ​

厌倦了枯燥的英语听写训练？试试 English Listen —— 一款让你爱上英语学习的现代化听写工具！

项目介绍 ​

项目概述 ​

English Listen 是一款专为英语学习者设计的智能听写软件。基于 C++17 和 Qt6 开发，支持 Windows 平台。它不仅仅是一个单词播放器，更是你提升听力和拼写技能的秘密武器。无论你是学生、教师还是自学爱好者，它都能帮你轻松搞定英语听写。

核心功能 ​

  • 智能词库管理：支持灵活编辑、多级分类、批量操作和自动保存
  • 个性化训练控制：自由调节朗读间隔、暂停/继续、快速导航、重复播放
  • 视觉体验优化：深色/浅色主题切换、系统主题适配、流畅动画效果
  • 学习进度追踪：成绩记录、数据可视化图表、多维度分析、历史管理
  • 双语音引擎：支持 Windows SAPI 和 Flite 第三方引擎，离线可用
  • 智能听写模式：纸笔模式和在线模式灵活切换
  • 快捷键支持：空格键重复、方向键切换、ESC暂停等便捷操作

使用场景 ​

  • 学生：课后英语听写练习，提升听力理解和拼写能力
  • 教师：课堂听写测试工具，支持纸笔和在线两种模式
  • 自学者：自主学习英语，通过可视化图表追踪学习进度
  • 培训机构：作为教学辅助工具，提供专业的听写训练功能

安装与使用 ​

安装步骤 ​

  1. 下载安装包：从 下载地址 下载最新版本安装包
  2. 运行安装程序：双击 EnglishListenSetup.exe 启动安装向导
  3. 选择安装路径：按照向导提示选择安装目录（默认：C:\Program Files\EnglishListen）
  4. 完成安装：点击"安装"按钮，等待安装完成
  5. 首次运行：安装完成后勾选"运行 English Listen"启动程序

快速开始 ​

  1. 启动程序：双击桌面快捷方式或从开始菜单启动
  2. 添加单词：点击"查看单词"按钮，在文本框中输入要练习的单词（每行一个）
  3. 开始测试：点击"开始听写测试"，选择听写模式（纸笔模式或在线模式）
  4. 听写练习：根据语音提示写下听到的单词
  5. 查看答案：测试结束后点击"查看答案"核对结果

高级配置 ​

  • 语音引擎设置：在设置对话框中选择 SAPI 或 Flite 引擎
  • 时间间隔调节：在测试界面调整朗读间隔（1-10秒）
  • 主题切换：使用右键菜单切换深色/浅色主题
  • 快捷键：测试界面支持空格键重复、方向键导航、ESC暂停
